¿Cómo configuro el token para ver repositorios privados?

1. Haz clic en el enlace de arriba ("Fine-grained Token").

2. Ponle un nombre y elige su duración.

3. En Repository access, elige "All repositories" o "Only select repositories" (y selecciona los privados que necesites).

4. Despliega Repository permissions, busca Contents y ponlo en "Read and write".

5. Baja del todo y dale a Generate token. Copia y pega aquí el código que empieza por github_pat_.

💡 Nota: El token nunca sale de tu navegador web.
