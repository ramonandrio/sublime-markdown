Ajustes

Integración con Coda

Genera tu API Key en coda.io/account → sección "API". Los documentos se cargarán de forma privada sin necesidad de publicarlos.

Integración con Notion

Crea una integración en notion.so/my-integrations y copia el "Internal Integration Secret". Recuerda que debes compartir tus páginas con la integración desde Notion para que aparezcan aquí.

Modelo local (Ollama)

Conecta tu Ollama local para usar modelos como llama3 o mistral desde un terminal integrado. Instala Ollama desde ollama.com y arráncalo antes de detectar.
